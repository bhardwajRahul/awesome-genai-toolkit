# 🧑‍🏫 Agent Framework Guides

> Crash courses and deep dives into the most popular AI agent frameworks — learn to build agents with any stack.

## 📂 Structure

```
agent-skills/frameworks/
├── google-adk/           # Google Agent Development Kit
├── openai-agents-sdk/    # OpenAI Agents SDK
├── crewai/               # CrewAI multi-agent framework
├── langgraph/            # LangGraph stateful agent orchestration
├── autogen/              # Microsoft AutoGen
└── phidata/              # Phidata agent toolkit
```

## 🔥 Why This Is Trending

The agent framework landscape is evolving rapidly. Google ADK, OpenAI Agents SDK, CrewAI, LangGraph, and AutoGen each offer different paradigms — from simple function-calling agents to complex multi-agent swarms. Developers need hands-on guides to pick the right framework and get productive fast.

## 🚀 Getting Started

```bash
cd agent-skills/frameworks/<framework-name>
pip install -r requirements.txt
python starter.py
```
