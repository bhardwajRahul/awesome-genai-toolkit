## 🧠 What You'll Learn

- **Multi-vector retrieval** for images, tables, and text combined
- Using **GPT-4o / Gemini** as the multimodal generation backbone

## 🖼️ Why Vision RAG?

Most enterprise documents are not just text:

- 📊 Charts and graphs that OCR destroys
- 📄 Scanned PDFs with layouts that matter
- 🗺️ Diagrams that carry meaning in structure

Vision RAG retrieves the **visual page** and asks the LLM to reason over it directly.

## 🔗 Key Concepts

- **ColPali** — page-level visual embeddings
- **LLaVA / GPT-4V** — vision-language generation
- **Image Embeddings** — CLIP, BLIP-2 for visual similarity
- **Multi-modal Vector Stores** — storing image + text embeddings together
