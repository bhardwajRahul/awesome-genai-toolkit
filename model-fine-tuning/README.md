# ⚡ Model Fine-Tuning

> Customize LLMs for your use case — LoRA, QLoRA, RLHF, and full fine-tuning recipes with step-by-step guides.

## 📂 Structure

```
model-fine-tuning/
├── lora/                 # Low-Rank Adaptation fine-tuning
├── qlora/                # Quantized LoRA for consumer GPUs
├── full-fine-tune/       # Full parameter fine-tuning
├── rlhf/                 # Reinforcement Learning from Human Feedback
└── datasets/             # Curated datasets for fine-tuning
```

## 🔥 Why This Is Trending

Fine-tuning is how you make a generic LLM into YOUR LLM. With LoRA and QLoRA, you can fine-tune models on a single GPU for free on Google Colab. Combined with open-weight models like Llama 3, Gemma, Mistral, and Qwen, fine-tuning has never been more accessible or powerful.

## 🚀 Getting Started

```bash
cd model-fine-tuning/<project-name>
pip install -r requirements.txt
python train.py
```
