# 📦 Open-Source GenAI Tools

> A curated directory of the best open-source libraries, SDKs, and developer tools powering the GenAI ecosystem.

## 📂 Structure

```
oss-genai-tools/
├── agent-frameworks/     # CrewAI, LangGraph, AutoGen, Swarm, etc.
├── llm-sdks/             # OpenAI SDK, Anthropic SDK, Google GenAI, etc.
├── vector-databases/     # ChromaDB, Pinecone, Weaviate, Qdrant, etc.
├── prompt-tools/         # DSPy, Instructor, Outlines, BAML, etc.
└── deployment/           # vLLM, Ollama, LitServe, TGI, etc.
```

## 🔥 Why This Is Trending

The open-source GenAI ecosystem is exploding — new libraries ship every week. Developers need a curated, opinionated guide to pick the right tools for their stack. This folder serves as a living directory with reviews, comparisons, and quick-start examples.
